# ARKTIK Brand Assets

This repository contains official ARKTIK brand assets (logos and related marks) for **approved, referential use**.

**Important:** This repo is public for ease of access, but the assets are **not** open source. Use is governed by the license in this repository and ARKTIK trademark rules.

## What’s in here
- Official ARKTIK logos (PNG/SVG where available)
- Optional: wordmarks, lockups, approved color references

## Allowed uses (referential)
You may use these assets **only** to:
- reference ARKTIK in press/editorial coverage
- link to ARKTIK’s official properties
- identify ARKTIK as a partner/sponsor **only when a real relationship exists**
- display “powered by / supported by” only with written approval (if required by your agreement)

## Prohibited uses (hard no)
You may not:
- claim or imply ARKTIK endorsement, affiliation, or sponsorship where it does not exist
- use the assets in deceptive, illegal, defamatory, or harmful contexts
- register confusingly similar names, domains, or marks
- alter the logo design (colors, proportions, elements, orientation), except for **resizing** or **format conversion**
- use ARKTIK marks to brand a separate entity (including any non-ARKTIK lanes)
- use these assets as part of any product/service name, app icon, or company identity without written permission

## Modification rules
Allowed:
- resizing
- file format conversion (e.g., SVG → PNG)
- background removal for placement (without changing the mark)

Not allowed:
- recoloring
- re-drawing
- adding effects/filters
- changing proportions
- combining with other marks to form a new logo

## Attribution (recommended)
When feasible: “ARKTIK®” and/or “ARKTIK Education®” as appropriate.

## Requests / permissions
If you need usage beyond “referential use” (advertising campaigns, co-branding, merch, product packaging, app icons), you must obtain written permission from ARKTIK.

Use the contact channel specified in your ARKTIK agreement or ARKTIK’s official website:
- https://arktik.education

## Lane separation note
This repo is **ARKTIK-only**. It does not contain assets for other lanes/entities.
